# :dna: Ito Lab Startup: Selection-exam

## :pushpin: 概要
本リポジトリは、名古屋大学 井藤彰研究室発スタートアップの「全職種共通」採用試験会場です。

私たちのチームには、従来の「事務」「営業」「開発」という壁はありません。
CEOが研究と経営を両立させるため、すべてのメンバーには**「情報のデジタル化」「非同期コミュニケーション」「AIによる自動化」**の素養を求めます。

---

## :hammer_and_wrench: 必須スキルセット（Level 0）
以下のツールが使えると採用の可能性が大きく上がります。

* **Terminal:** `wezterm` 等のモダンなターミナル環境
* **Version Control:** `GitHub` (CLIベースでの操作)
* **AI Tooling:** `geminicli` をはじめとするLLM活用能力
* **Documentation:** `Markdown` による構造化テキストの作成

---

## :memo: 試験フロー

### 第1課題：自己PRリポジトリの構築
1. 本リポジトリを `Fork` してください。
2. `branch`を切り、以下の内容を含むMarkdownファイルを作成し、`Push` して,`PullRequest`を送ってください。
   - あなたの強み（職種不問）
   - なぜこの「技術優先」の環境で働きたいのか
   - 私たちの技術（磁性ナノ粒子等）をどう社会に広めるべきか、あなたの考え

### 第2課題：仕様変更（PRベースの追記）
1. 最初のPRを出した後、CEOから いくつかの質問が投げられます。
2. その質問を踏まえてリポジトリ内にある`pptxgenjs`のテンプレートを用いて`プログラムで`自己PR資料を作成してください。markdownでは書ききれなかった情報を載せても大丈夫です。もちろんLLMは自由に使用してください。
3. すべてのやり取りは GitHub の `Pull Request` 上のコメントで行われます。

## :scales: 選考基準
* **自律性:** 公式ドキュメントを読み込み、自力で環境構築と実装ができるか。
* **論理:** 文章およびコードに論理性があるか。
* **適応:** CEOからの急な「仕様変更（PRでの指示）」に即座に対応できるか。
* **理解**: 私たちの研究や業務の内容を深く理解できているか。

## :gem: 私たちが約束すること
この「Selection-exam」を突破したメンバーには、以下の環境を約束します。

1.  **圧倒的な効率:** 無駄な会議や、メールでのファイル送受信から解放されます。
2.  **最先端の研究環境:** 井藤研究室の知見を間近で学びながら、社会実装に携わる興奮。
